Hondo
=======

Open source 3D engine using OpenGL and C++.
=======

To build on linux:
  * Install dependencies
  * run cmake, followed by make.
  * run Hondo

Dependencies:
  * cmake
  * glfw
  * glew-1.10.0
  * glm
  * Boost_1_55_0
    * regex
  * SOIL, with [this](https://bugs.debian.org/cgi-bin/bugreport.cgi?bug=727175) patch
